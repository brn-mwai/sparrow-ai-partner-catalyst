## TAM - Total Addressable Market

### Definition
Every salesperson who makes phone/video calls and could benefit from AI-powered practice.

### Calculation

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   TAM CALCULATION                                                           │
│   ═══════════════                                                           │
│                                                                             │
│   US B2B Salespeople who make calls:           5,700,000                    │
│                                                                             │
│   × Potential spend on AI training tool:       × $1,000/year                │
│                                                                             │
│   ─────────────────────────────────────────────────────────                 │
│                                                                             │
│   US TAM:                                      $5.7 BILLION                 │
│                                                                             │
│                                                                             │
│   BREAKDOWN BY SEGMENT:                                                     │
│   ──────────────────────                                                    │
│                                                                             │
│   Inside Sales Reps (2.6M × $1,000)          = $2.6B                        │
│   Field Sales Reps (3.1M × $1,000)           = $3.1B                        │
│   ─────────────────────────────────────────────────────                     │
│   Total US TAM                               = $5.7B                        │
│                                                                             │
│                                                                             │
│   GLOBAL EXPANSION (2.5× multiplier):                                       │
│   ─────────────────────────────────────                                     │
│                                                                             │
│   Global TAM                                 = $14.3B                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Why $1,000/year per rep?
- Companies already spend $2,000/year on sales training per rep
- AI training tools represent a fraction of total training budget
- $1,000/year = $83/month — reasonable for productivity tool
- Comparable to: Gong ($100+/user/mo), Outreach ($100+/user/mo)

## SAM - Serviceable Addressable Market

### Definition
The segment we can realistically serve with our current product: **Inside sales reps who do cold calls, discovery calls, and need to handle objections** — primarily in English-speaking markets.

### Calculation

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   SAM CALCULATION                                                           │
│   ═══════════════                                                           │
│                                                                             │
│   US Inside Sales Reps:                        2,600,000                    │
│                                                                             │
│   × Our average price point:                   × $600/year                  │
│                                                                             │
│   ─────────────────────────────────────────────────────────                 │
│                                                                             │
│   US SAM:                                      $1.56 BILLION                │
│                                                                             │
│                                                                             │
│   WHY INSIDE SALES ONLY:                                                    │
│   ──────────────────────                                                    │
│                                                                             │
│   ✓ Make high volume of calls (perfect for practice)                        │
│   ✓ Cold calling is core skill                                              │
│   ✓ Objection handling is daily reality                                     │
│   ✓ Discovery calls are standard workflow                                   │
│   ✓ Work remotely (can practice anytime)                                    │
│                                                                             │
│   ✗ Field sales = fewer calls, more relationship-based                      │
│   ✗ Retail sales = different skill set                                      │
│                                                                             │
│                                                                             │
│   SEGMENT BREAKDOWN:                                                        │
│   ──────────────────                                                        │
│                                                                             │
│   SDRs/BDRs (lead gen):        800,000 × $600  = $480M                      │
│   Inside AEs (closing):      1,200,000 × $600  = $720M                      │
│   Sales managers (coaching):   600,000 × $600  = $360M                      │
│   ────────────────────────────────────────────────────                      │
│   Total US SAM                                 = $1.56B                     │
│                                                                             │
│                                                                             │
│   ENGLISH-SPEAKING MARKETS (1.5× US):                                       │
│   ────────────────────────────────────                                      │
│                                                                             │
│   Total SAM (US + UK + Canada + Aus)           = $2.3B                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Why $600/year average?
- Blended average of our tiers
- Starter ($348) + Pro ($948) + Team ($588) ÷ 3 ≈ $600
- Lower than TAM assumption because we're more accessible than enterprise tools
- Accounts for mix of individual users and team licenses

### Why 10% Penetration is Realistic

| Factor | Justification |
|--------|---------------|
| **New category** | AI voice training is emerging — early movers capture share |
| **Clear ROI** | 1 closed deal pays for year of Sparrow |
| **Viral potential** | Reps share with teams, teams share with companies |
| **Low switching cost** | No integration required, easy to try |
| **Pain is acute** | 91% miss quota — buyers are desperate |

### Why SaaS/Tech First?

1. **Early adopters** — Embrace new tools faster
2. **High call volume** — SDRs make 50-100 calls/day
3. **Budget exists** — Already spend on sales tools
4. **Remote-first** — Can practice anytime
5. **Measurable** — Track improvement in quota attainment

## Comparable Company Benchmarks
