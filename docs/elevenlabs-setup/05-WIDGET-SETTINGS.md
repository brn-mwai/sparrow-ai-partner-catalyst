# ElevenLabs Agent - Widget Settings

Configure the widget interface options.

---

## Step 1: Go to Widget Tab

1. Click the **"Widget"** tab at the top of the page
2. (It's between "Tools" and "Security")

---

## Step 2: Configure Settings

Set each option as shown:

### Feedback Collection
```
✅ ON
```
Callers can rate satisfaction after the conversation.

### Widget v2 (Beta)
```
❌ OFF
```
Keep this off for now.
