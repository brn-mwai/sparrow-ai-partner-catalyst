# ElevenLabs Agent - Publish and Copy Credentials

Publish your agent and get the credentials.

---

## Step 1: Publish the Agent

1. Click the **"Publish"** button (top right, blue button)
2. Wait for it to change to **"Live"** status
3. You should see a green "Live 100%" badge

---

## Step 2: Copy Agent ID

1. Look at the top of the page, near the agent name
2. You'll see: `ID: ...xxxxx` (abbreviated)
3. **Click on it** to copy the full ID
4. The full ID looks like: `agent_4001kd5v9vwgfb385mpnh3wvtt69`

---

## Step 3: Get API Key

1. Click your **profile icon** (top right corner)
2. Click **"Profile + API key"**
   - Or go directly to: https://elevenlabs.io/app/settings/api-keys
3. Click **"Create API Key"** (or copy existing)
4. Copy the key (starts with `sk_`)

### Save your API Key:
```
sk_________________________________
```
(Write it here or paste somewhere safe)

---

## Step 4: Verify Everything

Before continuing, make sure you have:

- [ ] Agent is showing **"Live"** status
- [ ] Agent ID copied (starts with `agent_`)
- [ ] API Key copied (starts with `sk_`)
